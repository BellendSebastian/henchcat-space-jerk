core/Renderer~ Renderer

new Renderer(scene)

Renderer class. Accepts a THREE.Scene (optional) and handles creating generic lights and cameras. No scene data will be played with here, however, that should be stored in separate scene classes and provided to the Renderer on instantiation or using the changeScene method.
